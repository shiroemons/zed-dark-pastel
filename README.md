# Dark Pastel for Zed

A dark theme inspired by iTerm2's Dark Pastel color scheme, featuring vibrant pastel colors on a pure black background.

## Installation

### Via Extensions (when published)

Search "Dark Pastel" in Zed extensions.

### Manual

Copy the theme folder to `~/.config/zed/themes/`

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Background | #000000 | Main background |
| Foreground | #FFFFFF | Text |
| Red | #FF5555 | Strings, errors |
| Green | #55FF55 | Properties, success |
| Yellow | #FFFF55 | Keywords, warnings |
| Blue | #4488FF | Functions |
| Magenta | #FF55FF | Constants, numbers |
| Cyan | #55FFFF | Types, info |
